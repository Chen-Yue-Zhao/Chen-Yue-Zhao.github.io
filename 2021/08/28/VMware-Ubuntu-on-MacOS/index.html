VMware & Ubuntu Installation on MacOS

系统配置

MacBook Pro (13-inch, 2019, Four Thunderbelt 3 ports, 8 GB Memory)
macOS Bug Sur Version 11.2.3

安装步骤

  1. 官网下载VMware Fusion
  2. 官网下载最新版本ubuntu
  3. Vmware Fusion创建新的虚拟机

1.官网下载VMware Fusion

我并没有选择在官网下载，原因有三。
第一，我没太看懂官网的下载地址与下载要求；
第二，我注意到VMware是收费产品（注意到另一个虚拟机软件年年收费，在乎性价比的可以注意）
第三，北大官方购买的了正版VMware。

在官网下载dmg后安装VMware。在安装过程中会要求输入密钥，这个密钥通过正规渠道完成购买后是能够在官网得到查询的。
p.s. baidu应该也能费功夫找到可以用的密钥。

2.官网下载最新版本ubuntu

进入官网https://www.ubuntu.com/download/desktop下载最新版本。

本次下载选用Ubuntu 20.04.3 LTS，下载成功后，会得到一个.iso文件。注意不要把.iso文件解压了。
接下来将.iso文件放置在你需要的文件路径上。我选用的是/User/Developer这个路径，方便接下来寻找。


3. Vmware Fusion创建新的虚拟机

打开VMware Fusion，选择File - New，在界面内选择Create a custom virtual machine

在选择系统时候找到Linux - Ubuntu 64-bit，然后一路continue下去。
